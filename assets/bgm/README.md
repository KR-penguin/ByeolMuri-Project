# BGM 안내

이 폴더에 단일 BGM 파일을 추가하면 레벨 진입 시 자동으로 재생됩니다.

필요한 파일명 (프로젝트에서 기대하는 이름, 선택 사항):
- `경쾌한 BGM.mp3` 또는 `경쾌한 BGM.wav` — 레벨 진입 시 반복 재생되는 단일 BGM

자동 생성: 파일이 없으면 게임이 시작될 때 기본 경쾌한 멜로디 WAV 파일(`경쾌한 BGM.wav`)을 자동으로 생성하여 재생합니다.

권장 형식: `.mp3` 또는 `.wav` (프로그램은 WAV 자동생성이 가능)

설치 방법(수동):
1. 위에 적힌 이름으로 파일을 준비합니다 (저작권에 주의하세요).
2. `assets/bgm/` 폴더에 파일을 넣습니다.
3. 게임을 실행하고 레벨을 로드하면 해당 BGM이 무한 반복 재생됩니다.

참고: 파일이 없으면 콘솔에 안내 메시지가 출력되며 기본 BGM이 자동으로 생성/재생됩니다.
